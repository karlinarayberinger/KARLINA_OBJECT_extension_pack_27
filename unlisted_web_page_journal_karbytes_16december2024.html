darknet server tests

* * *

On 16_DECEMBER_2024, karbytes implemented some tests to verify that karbytes’ first .onion web page was publicly accessible (to the World Wide Web via The Onion Router (Tor) network) while being served from karbytes’ Lenovo laptop in karbytes’ closet (and while that laptop was connected to an electrical outlet in the wall of that closet’s encompassing room and to the Wi-Fi network whose router was in the west-most corner of that room’s encompassing house).

First karbytes overwrote the hard drive of karbytes’ HP laptop computer with a fresh installation of Ubuntu from karbytes’ bootable flash drive (which was configured by karbytes to allow users to test Ubuntu or to install Ubuntu).

Then karbytes installed the Tor browser on that HP laptop in order to visit karbytes’ first .onion web page which is described in more detail at the following web page: https://karbytesforlifeblog.wordpress.com/journal_karbytes_10december2024/

Finally, karbytes determined that the .onion web page was publicly accessible by testing the server-client connection from various remote Wi-Fi networks (namely the Peet’s Coffee in Castro Valley, California and the Whole Foods Market in Dublin, California) on a server instance lasting more than seven hours (due to karbytes activating the Caffeine background application which prevents the computer from entering sleep mode even while that computer is left idle for an indefinitely long time period (and the details about how karbytes discovered, installed, and used that Caffeine application are available in the following web page: https://karbytesforlifeblog.wordpress.com/chatgpt_karbytes_14december2024/)).

(Note that karbytes turned on (and left on at a relatively low volume) an indefinitely-playing sound generator web page application at the following Uniform Resource Locator on the Lenovo laptop serving the .onion web page in order to signify that the server was turned on and not in sleep mode (which suggests that the .onion web page is accessible across the public World Wide Web): https://mynoise.net/NoiseMachines/isochronicBrainwaveGenerator.php).
